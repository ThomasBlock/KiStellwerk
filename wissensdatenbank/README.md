# Wissensdatenbank

Zentrale Wissensbasis von KiStellwerk.

## Bereiche

- **ki-grundlagen/** - Grundlagen zu KI, LLMs, Machine Learning, Deep Learning
- **tools-und-plattformen/** - Dokumentation und Bewertungen von KI-Tools und Plattformen
- **prompting/** - Prompt Engineering Techniken, Templates und Beispiele
- **branchenwissen/** - Branchenspezifisches Wissen fuer Kundenprojekte
- **best-practices/** - Bewaehrte Methoden, Standards und Richtlinien
