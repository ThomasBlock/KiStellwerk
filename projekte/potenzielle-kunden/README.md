# Potenzielle Kunden

Zentrale Liste und Informationen zu potenziellen Kunden (Leads).

## Struktur

Pro potenziellem Kunden kann ein eigener Unterordner angelegt werden mit:
- Kontaktdaten und Ansprechpartner
- Notizen aus Erstgespraechen
- Bedarfsanalyse / Potenzialeinschaetzung
- Status im Vertriebsprozess

## Status-Kategorien

- **Neu** - Erster Kontakt / Lead identifiziert
- **In Kontakt** - Erstgespraech gefuehrt
- **Angebot** - Angebot wurde erstellt
- **Verhandlung** - In Vertragsverhandlung
- **Gewonnen** - Wird zu aktivem Kunden (verschieben nach `kunden/`)
- **Pausiert** - Aktuell kein Bedarf, spaeter erneut kontaktieren
