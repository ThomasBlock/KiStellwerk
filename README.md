# KiStellwerk

Zentrale Datenstruktur der KI-Agentur **KiStellwerk**.

## Ordnerstruktur

```
KiStellwerk/
|
|-- wissensdatenbank/          # Zentrale Wissensbasis
|   |-- ki-grundlagen/         # KI-Konzepte, LLMs, Machine Learning
|   |-- tools-und-plattformen/ # Dokumentation zu KI-Tools (OpenAI, Anthropic, etc.)
|   |-- prompting/             # Prompt Engineering, Techniken, Beispiele
|   |-- branchenwissen/        # Branchenspezifisches Wissen fuer Kundenprojekte
|   |-- best-practices/        # Bewehrte Methoden und Standards
|
|-- projekte/                  # Projektmanagement
|   |-- kunden/                # Kundenprojekte (pro Kunde ein Unterordner)
|   |-- intern/                # Interne Projekte
|   |-- vorlagen/              # Projektvorlagen und Blueprints
|
|-- services/                  # Dienstleistungsangebote
|   |-- beratung/              # KI-Beratung und Strategie
|   |-- entwicklung/           # KI-Entwicklung und Integration
|   |-- schulungen/            # Workshops und Trainings
|   |-- automatisierung/       # Prozessautomatisierung mit KI
|
|-- templates/                 # Dokumentenvorlagen
|   |-- angebote/              # Angebotsvorlagen
|   |-- vertraege/             # Vertragsvorlagen
|   |-- praesentationen/       # Praesentationsvorlagen
|   |-- berichte/              # Berichtsvorlagen
|
|-- prozesse/                  # Unternehmensprozesse
|   |-- onboarding/            # Kunden- und Mitarbeiter-Onboarding
|   |-- projektmanagement/     # PM-Methoden und Workflows
|   |-- qualitaetssicherung/   # QA-Prozesse und Checklisten
|
|-- marketing/                 # Marketing und Vertrieb
|   |-- website/               # Website-Inhalte
|   |-- social-media/          # Social-Media-Plaene und Content
|   |-- content/               # Blog, Newsletter, Fachartikel
|   |-- case-studies/          # Fallstudien und Erfolgsgeschichten
|
|-- team/                      # Team und Organisation
|   |-- rollen/                # Rollenbeschreibungen und Zustaendigkeiten
|   |-- weiterbildung/         # Lernressourcen und Entwicklungsplaene
|
|-- finanzen/                  # Finanzen (vertraulich)
|   |-- buchhaltung/           # Rechnungen und Belege
|   |-- controlling/           # KPIs und Finanzberichte
|
|-- tools/                     # Interne Tools und Skripte
    |-- scripts/               # Automatisierungsskripte
    |-- configs/               # Konfigurationsdateien
```

## Nutzung

Jeder Bereich hat eine klare Zustaendigkeit. Neue Inhalte werden im passenden Ordner abgelegt. Bei Unsicherheit: `wissensdatenbank/` ist der Startpunkt fuer Wissen, `projekte/` fuer alles Projektbezogene.
